## General
### This mod adds a lot of tires:

- (10x10x3) huge tire
- (10x10x4) huge wide tire
- (12x12x4) huge rowing tire (can row, floats)
- (12x12x6) huge rowing wide tire (can row, floats)
- (3x3x1) big tractor tire
- (2x2x1) tractor tire
- (3x3x1.5) truck tire
- (2x2x1) small truck tire
